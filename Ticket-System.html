About the project

This is my first and most complex project up to that moment done purely in C++.
It was presented as the "OOP" final project in Computer Engeneering and it got a 10/10 final score.
Later, it was used by my family for a small business.

It is a CRUD application to list products scanned by a code bar scanner.

As a CRUD, you can create, read, update and delete them.

Also, you can administrate your client list, order them and search a specific one.

When you start making the products list, with the first scanned (or passed by code) product, you will get above the list the ticket number assigned to that buy.

When you finish it, you can either save it (Guardar) or print it (Imprimir).

- The button, will save the list in a ticket history.

- The second one, will do the same as the previous button, plus create a .txt file with all the details in \debug.w32.
