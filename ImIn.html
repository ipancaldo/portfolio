About the project -> UNDER CONSTRUCTION

This is a small project made with Angular and Firebase.
It helps organize who is going to assist to a certain event.
It is currently under development.

In order to be able to use this app, you must create an user and it will be stored in Firebase Realtime Database.

Once you have your user, you will need to log in.

Then, you will be able to do a few things in the app:

- Create a new unique event to check who's going to assist.

- Create a new daily/weekly/monthly event to check who's going to assist.

- Confirm that you will assist to an invitation.
